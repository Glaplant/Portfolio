Ch2

  • well commented code is the hallmark of a ninja programmer
  • best practice to write each statement on a new line, terminated by a semi-colon
  • A block is a series of statements that are collected together inside curly braces
  • Primitive Data Types:String, Symbol, Number, Boolean, Undefined, Null
  • Any value that isn’t one of the primitive data types listed above is an object. These include arrays, functions and object literals...
  • a special operator called typeof for finding out the type of a value
  • Variables have to be declared before they can be used.
  • JavaScript uses the keywords const and let to declare variables.
  • const is used when the variable will not be reassigned to another value
  • let is used if the variable might be reassigned later in the program
  • try to use const to declare most variables. This helps to avoid any bugs caused by unexpected changes in assignment.
  • A ninja programmer will try to keep the number of global variables to a minimum, because any variables that share the same name will clash and potentially overwrite each other’s values
  • When naming constants and variables, you should try to give them sensible names that describe what the variable represents
  • JavaScript’s built-in functions use the camel-case notation and this is probably the best convention to follow when naming the constants and variables in your code. The most important thing is to be consistent.
  • If you want to use double quote marks inside a string literal, you need to use single quote marks to enclose the string. And if you want to use an apostrophe in your string, you will need to use double quote marks to enclose the string, otherwise the apostrophe will terminate the string, causing an error:
  • Another option is to escape the quotation mark. You place a backslash before the apostrophe so it appears as an apostrophe inside the string instead of terminating the string:
  • A method is an action that a primitive data type or object can perform. To call a method, we use the dot operator (.) followed by the name of the method, followed by parentheses
  • Template literals use the backtick character, ` , to deliminate the string
  • They also allow interpolation of JavaScript code. This means that a JavaScript expression can be inserted inside a string and the result will be displayed
  • Template literals can be thought of as super-powered strings as they behave in the same way as normal string literals, but with the extra power of string interpolation. For this reason, it is not uncommon to see backticks used to create all strings in ES6 code.
  • The main use-case for symbols is as object property keys
  • The toFixed() method rounds a number to a fixed number of decimal places
  • Number.isFinite() method. This will return true if the value is a number that isn’t Infinity, -Infinity or NaN:
  • Type coercion happens when the operands of an operator are of different types. In this case, JavaScript will attempt to convert one operand to an equivalent value of the other operand’s type.
  • If you need to change the type, then do it manually, rather than relying on type coercion to do it for you.
  • The best way to change a string to a number is to use the Number method. This will convert the string form of a number into an actual number:
