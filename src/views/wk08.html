CH8 HTML & CSS for the Real World

Transforms

  • Translation
  • Scaling
  • Rotation
  • Skew
  • Transform-origin

Translation is similar to position relative. Translate(x,y); translatex; translatey;

Scaling: scale(x,y); x horizontal scale; y vertical scale; scalex; scaley; Default origin point is the center.

Rotation: rotate(x deg ); Default origin point is the center.

Skew: skew(xdeg,ydeg); Skews along x and y axis; skewx(); skewy(); If second parameter is omitted, it will only skew the xaxis; Default origin point is the center.

Transform-origin; defaults to center; transform-origin: 0 0;

Transitions

Transitions; Allows values to change over time

  • Step 1: Declare original state.
  • Step 2: Declare final state.
  • Step 3: Include the transition functions
    • transition-property
    • transition-duration
    • transition-timing-function
      • ease
      • linear
      • ease-in
      • ease-out
      • ease-in-out
      • cubic-bezier
    • transition-delay

Transition shorthand;The transition property is shorthand for the four transition functions. transition: value 1 value 2 value 3 value 4;

  • Value 1: transition-property
  • Value 2: transition-duration
  • Value 3: transition-timing-function
  • Value 4: transition-delay

Multiple Transitions;
transition-property: transform, color;
shorthand;
transition: color 0.2s ease-out, transform 0.2s ease-out
