CH8 HTML & CSS for the Real World

Translation is similar to position relative. Translate(x,y); translatex; translatey;

Scaling: scale(x,y); x horizontal scale; y vertical scale; scalex; scaley; Default origin point is the center.

Rotation: rotate(x deg ); Default origin point is the center.

Skew: skew(xdeg,ydeg); Skews along x and y axis; skewx(); skewy(); If second parameter is omitted, it will only skew the xaxis; Default origin point is the center.

Transform-origin; defaults to center; transform-origin: 0 0;
